## Overview

This repository provides a Terraform configuration for managing infrastructure as code. It includes modules for creating and managing AWS resources.

## Prerequisites

* Terraform version 1.1.0 or later
* AWS CLI version 2.0.0 or later
* AWS account with necessary permissions

## Usage

1. Clone the repository: `git clone https://github.com/your-username/infra-terraform.git`
2. Navigate to the repository: `cd infra-terraform`
3. Initialize the Terraform working directory: `terraform init`
4. Create an AWS provider configuration file: `terraform apply`
